GitHub + Hexo 搭建个人博客

个人博客搭建（Windows 10）

环境安装

nodejs下载安装

  • 在cmd或者Git里键入
    ‘’’
    node -v
    npm -v
    ‘’’
    检查是否安装完毕
  • 使用npm安装cnpm
    ‘’’
    npm install -g cnpm –registry=https://registry.npm.taobao.org
    cnpm -v
    ‘’’

    使用 cnpm 安装 hexo

    ‘’’
    cnpm install -g hexo-cli
    ‘’’
  • 检查hexo是否安装完毕
    ‘’’
    hexo -v
    ‘’’
  • 创建Blog文件夹
    ‘’’
    mkdir blog
    cd blog
    hexo init # 初始化博客
    hexo s # 启动本地服务器
    ‘’’
    访问localhost:4000
    创建第一篇文章
    ‘’’
    hexo new “我的第一篇Hexo博客文章”
    ‘’’
    返回blog目录
    ‘’’
    hexo clean
    hexo g
    ‘’’

    Github创建新仓库

  • 切记仓库名为Username.github.io
    ‘’’
    cnpm install –save hexo-deployer-git
    ‘’’
  • 修改 _config.yml 文件

‘’’

Deployment

Docs: https://hexo.io/docs/deployment.html

deploy:
type: git
repo: https://github.com/YourGithubName/YourGithubName.github.io.git
branch: master
‘’’

‘’’
hexo d
‘’’

  • 访问Username.github.io

推送指南

‘’’
hexo clean # 清理一下
hexo g # 生成
hexo d # 部署到远端
‘’’
