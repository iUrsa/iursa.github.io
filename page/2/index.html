我在方洋就业了~

2020-08-02

昨天是2020年建军节，离开学校马上一个月了，又迎来了新的一个月。
7月11日，我走进了方洋。公司给我们安排了为期两周的入职培训。
我感觉我很幸运，被分到了三班，命名为“菁英三班”。我们班有44个人，班级同学来自于全国各地，各位硕士生毕业于C9等高等学府。我望尘莫及，初入班级尽带一丝自卑与惭愧。

刚从CCZU毕业，让大众难舍之情为我背锅就是难免有些留恋以至于没来得及过渡——即毕业后没来得及准备就到了公司。
最近总是一个人吃饭，我不知道自己会不会习惯。昨天傍晚，也就是8月1日黄昏，我在方言之家吃完饭，可能是饭的味道没有吸引我，或许只是因为我一个人吃饭的原因，这碗里的面条确实像是被白水煮的一样。我立马逃离这个犹如是一个火灾现场，使我窒息。出门随即给父亲打了电话，不喜欢一个人走路，不算是想家了，仅仅是不想一个人走路吧，想找个人来跟我聊天。步伐时快时慢，感觉前方有希望在召唤我，我不太喜欢这种感觉：眼看到的不能触手可及。挂完父亲的电话连忙拨通了兄长的电话。

今日，与校友一起去市里逛了一圈。回来的路上我整个人的心情有些好转，是因为遇到了菁英三班的同学，一番问询之后得知，他是一个人在市区逛了一天，我深深佩服，他是有多么强大的内心，要是我绝对不可能一个人逛gai的。戏剧性的还有就是我从上车就认错了人，喊这位哥哥名字并非眼前人，大之尴尬诶！

稍后我将会上传培训期间的照片作为留念。

Java

2020-05-24

The building is divided into aparments and an apartment is divided into rooms;
A command is like a room. In the Java programming language, a command can`t exist on its own. It`s part of a function (in Java,’functions’ are also called ‘methods’). A method is part of a class. In the other words, a class is divided into methods and methods are divided into commands.

So a class is an apartment building, a function/method is an apartment, and a command is a room.
